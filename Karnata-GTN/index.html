Credits & Font Information

  • Font name: Karnata-GTN
  • Scripts supported: Kannada, Latin
  • Project by: Sanchaya.org
  • Typeface designer: Arun C Kallappanavar
  • Font technology: Variable Font (wght axis)
  • Styles / Weights: Regular, Medium, SemiBold, Bold, ExtraBold, Black (400–900)
  • Font license: SIL Open Font License (OFL)
  • About: Karnata GTN is an open-source variable Kannada typeface inspired by the handwriting of G. T. Narayana Rao. Developed by Arun C Kallappanavar as part of the Sanchaya Initiative, the project transforms handwritten letterforms into a modern digital font while preserving their natural rhythm and personality. Supporting Kannada and Latin scripts with a variable weight axis from 400 to 900, Karnata GTN combines traditional handwriting aesthetics with contemporary typography, making it suitable for web, UI, publishing, branding, and digital communication.

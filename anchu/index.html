Credits & Font Information

  • Maintainer: Arun C Kallappanavar
  • Font license: SIL Open Font License (OFL)
  • About: Anchu is an open-source Kannada typeface created as a personal learning project to explore type design, font engineering, Unicode implementation, and OpenType development. Developed through self-study, experimentation, and trial-and-error, the project served as a foundation for understanding how digital fonts are designed, built, and shaped. While Anchu was never intended to be a production-ready typeface, it became an important stepping stone that contributed to the development of later Kannada typefaces, including more refined text, display, and variable font projects. The font remains publicly available as a record of that learning journey and the beginning of a broader exploration into Kannada typography.

I'm Arun CK

Namaskara — I’m Arun C K, a type designer focused on Kannada script and multilingual font systems. I design, engineer, and release open fonts with attention to readability, file size, and robust OpenType feature support.

My work combines careful drawing with engineering: I build families (including variable fonts), tune spacing, and implement OpenType features for complex Indic behaviors. I’ve been studying type design for several years and work daily on designing and testing glyphs, metrics and features to make Kannada typography more robust across devices.

Recent projects include GTN — a text-first Kannada family optimized for on-screen reading — and companion families for display and monospace needs. Most releases are published under the SIL Open Font License so they can be used freely and improved by the community.

Tools & workflow: FontLab, Python scripting for font automation, variable font exports, intensive testing in browsers and native apps.

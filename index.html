Kannada Fonts — Free & Open Source

A curated collection of Unicode-ready Kannada and Latin fonts by Arun C Kallappanavar.

This site documents and distributes free, open-source Kannada and Latin typefaces designed and engineered by Arun CK. Each font page includes specimen previews, download links (ZIP), license details, and technical notes.

Browse released Kannada fonts, view specimens, and download ZIP packages. All fonts on this site are released under open-source licenses; see each font page for specific license information.

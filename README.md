# type.arunck.com

A static website for free, open-source Kannada and Latin fonts by Arun C Kallappanavar.

## What this site is for

- Showcases Unicode-ready Kannada fonts and font specimens.
- Provides downloadable font pages for families such as `Anchu`, `Sriraju`, `Karnata Bandipur`, `Karnata German`, `Karnata GTN`, and `Karnata Wesleyan`.
- Includes an `About` page with author and project context.

## Who should use it

This site is useful for:

- designers, developers, and typographers searching for open-source Kannada fonts.
- people who want to preview Kannada and Latin font styles before downloading.
- users looking for license-friendly font resources for web and print projects.

## Project structure

- `index.html` — home page with overview and font collection.
- `about/` — author and project information.
- `anchu/`, `sriraju/`, `Karnata-Bandipur/`, `Karnata-German/`, `Karnata-GTN/`, `Karnata-Wesleyan/` — individual font pages.
- `css/` — styles used by the site.
- `js/` — interactive scripts.
- `assets/` — images and fonts used in the site.
- `partials/` — shared page fragments.
- `scripts/` — utility scripts such as `generate-sitemap.js`.
